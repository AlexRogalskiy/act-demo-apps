Timezone

This program will first find out client's local timezone offset through JavaScript code:

new Date().getTimezoneOffset()

Then it will send the offset data to the server by calling

$.post("/~/i18n/timezone", {offset: offset})

After server updated the user session with the client timezone offset data, it will refresh this page and display the offset data
